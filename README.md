##  Overview
This project demonstrates a complete penetration testing workflow using Metasploitable2.  
It covers reconnaissance, exploitation, password attacks, social engineering, malware basics, and system hardening.


##  Tools Used
- Metasploit Framework  
- Hydra  
- John the Ripper  
- Nmap  
- Burp Suite  
- Linux  


##  Methodology
Recon → Scanning → Exploitation → Post-Exploitation → Reporting


##  Key Highlights
- Gained reverse shell using Metasploit  
- Performed SSH brute-force attack using Hydra  
- Cracked password hashes using John the Ripper  
- Created phishing simulation page  
- Applied system hardening techniques
